📊 Mini CRM – Client Lead Management System

A full-stack Customer Relationship Management (CRM) application built using the MERN Stack to manage, track, and analyze client leads efficiently.

📌 Project Overview

This project is a Lead Management System designed to help businesses:

Organize and track customer leads

Monitor lead status and progress

Manage follow-ups and notes

Improve conversion rates

It simulates a real-world CRM used by sales and marketing teams.

🎯 Problem Statement

Businesses often struggle with:

 Managing leads manually (Excel, paper)
 
 Missing follow-ups
 
 No centralized system
 
 Poor tracking of customer interactions
 
💡 Solution

This CRM system provides:

✔ Centralized lead management

✔ Easy status tracking (New → Contacted → Converted)

✔ Notes & follow-up system

✔ Secure admin authentication

✔ Data visualization for better decision-making

✨ Features

🔹 Core Features

Lead Management (Add, Edit, Delete)

Lead Status Tracking with color indicators

Notes & Follow-up reminders

Secure Admin Login (JWT Authentication)

🔹 Advanced Features

📊 Dashboard with charts (Recharts)

🔍 Search & Filter leads

📄 Pagination (10 leads per page)

📝 Activity Logs (track all actions)

🔔 Notifications for follow-ups

📤 Export leads to CSV

🌙 Dark Mode support

👥 Role-based access control

⚙️ Installation & Setup

1️⃣ Clone Repository

git clone https://github.com/123lakshmi-majji/FUTURE_FS_02.git

cd FUTURE_FS_02

2️⃣ Setup Backend

cd server

npm install

npm run dev

3️⃣ Setup Frontend

cd client

npm install

npm run dev

🔐 Environment Variables

Create a .env file in the backend:

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

PORT=5000
